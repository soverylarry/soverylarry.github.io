Team Project 1 - Free to Me movie app

Mobile Friendly App that eases the search for a free movie viewing option

GitHub Repo

'Free to Me' is an application to provide the user with information regarding if the movie they're considering viewing is already available in one of the users' current video subscriptions. The user simply enters the movie they'd like to watch and they will be presented with viewing options that may already be free to them. Conversely it will return the condition if they do not having any free viewing options. Each movie requested will return with it recommended viewing choices based on their entry and related movie data.

5 Day Weather Forecast App Project

Current Weather and 5 Day Forecast along with UV ratings.

GitHub Repo

This app will allow the user to request a cities' current weather conditions and a five day extended forecast as well. The UV index will also be displayed for the current conditions and convey the severity via a color code. The app will also show recently researched cities in the search history that allows the user to click on to also produce current and future weather conditions.
